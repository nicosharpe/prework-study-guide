HTML

  • HTML stands for Hypertext Markup Language
  • The head element contains the metadata for the web application
  • The body element represents the visible content shown to the end user
  • The title element defines the title of the webpage
  • The meta element defines the metadata such as the character set description, key words, author and viewpoint
  • All elements start with an opening and closing tag

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is a computer language used for laying out and structuring webpages in HTML
  • CSS describes how HTML elements should be shown on screen
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is an Open Source, cloud-based platform used for version control
  • It allows developers to store, manage and collaborate on code
  • You can use SSH Keys to use github without entering a password every time you interact with it
  • You can use the Command Line Interface (CLI) called Gitbash to interact with git locally from your PC
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Javascript is a scripting language that allows you to add multimedia elements such as animations and videos to webpages using code
  • It is needed to make a lot of web pages functional
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
